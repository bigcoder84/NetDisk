# NetDisk
这是一个Java SE网盘项目
网盘特点：
  1. 实现了网盘的秒传功能，若当前用户上传的文件已经被其他用户上传过了，那么就不会重新上传，实现秒传的效果。
  2. 网盘实现了多级目录、目录切换、目录回退、新建文件夹、重命名、删除、上传、下载（待开发）等基本的文件操作。
